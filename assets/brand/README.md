# NyaNovel brand assets

Direction: **Latent Frame**

The mark uses four inward-facing image-frame corners around a central signal diamond. Its square silhouette is designed to remain readable at favicon size.

## Palette

- Signal Coral: `#F35F52`
- Carbon: `#171719`
- Warm White: `#FAF8F4`
- Secondary neutral, banner only: `#BDB9B4`

## Files

- `logo-horizontal-light.svg` — coral mark and carbon wordmark for light surfaces
- `logo-horizontal-dark.svg` — coral mark and warm-white wordmark for dark surfaces
- `mark-on-light.svg` — standalone mark for light surfaces
- `mark-on-dark.svg` — standalone mark for dark surfaces
- `icon.svg` / `icon-512.png` — square application and favicon source
- `github-social-banner.svg` / `github-social-banner.png` — 1280×640 social banner source and output

The SVG wordmark uses Liberation Sans Bold with Arial/sans-serif fallbacks. Convert the wordmark to outlines in your preferred vector editor before sending it to a print vendor. The PNG is the canonical social-banner output.
